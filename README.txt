Description: 

This repository contains code pertaining to Levin et al (in prep)

	- elmer/
		Scripts and templates for the ELMER/ICE model

		The ELMER/ICE model is run on the Google Cloud using a docker container w/ ELMER/ICE (https://hub.docker.com/r/nwrichmond/elmerice/) (Courtesy of N.W. Richmond)
		Saves data to a Google Cloud bucket



	- python/
		Functions for analyzing .vtu output from the ELMER/ICE model

	- matlab/
		Scripts for measuring shoulders using REMA data
